Cook \*(v)
Files for Download


You can obtain this software a number of ways, including as source code and as pre-compiled binaries.

Master Sources

You can download the following files from this web site:

File Name Description
cook-\*(v).README The README file from the tar distribution.
cook-\*(v).tar.gz The complete source.
cook-\*(v).tut.pdf a Cook Tutorial, in PDF format.
cook-\*(v).ug.pdf the Cook User Guide, in PDF format. This manual describes the contents of cookbooks.
cook-\*(v).rm.pdf the Cook Reference Manual, in PDF format.
cook-\*(v).ae The complete source, in aedist format, for developers.


Second Sources

  • There is a tutorial in Japanese available, written by kwatch.
  • Cook is also available as Debian packages from the Debian distribution.
  • Cook is also available as part of the FreeBSD ports collection in the devel/cook directory.
  • Cook is also available as part of the OpenBSD ports collection in the devel/cook directory.
  • Cook is also available as part of the NetBSD ports collection in the devel/cook directory. The maintainer is groo@netbsd.org


LaunchPad PPA

If you use Ubuntu Linux, there are pre-built packages available for installation using the normal sudo apt-get install method, from this package archive (PPA).

Instructions are available there for how to manually add the PPA to your /etc/apt/sources.list file, but the following commands are sufficient on recent versions of Ubuntu: 

sudo add-apt-repository ppa:pmiller-opensource/ppa
sudo apt-get update
sudo apt-get install cook

Aegis Repo

Cook is developed using Aegis SCM. The Aegis repository for this project is available.
